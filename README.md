# Seminar Registration App

Aplikasi Android untuk pendaftaran seminar mahasiswa yang dibuat sebagai tugas Ujian Tengah Semester (UTS) mata kuliah Pemrograman Mobile I.

## 📱 Fitur
- Login & Register
- Halaman Utama
- Form Pendaftaran Seminar
- Validasi Input (Real-time)
- Dialog Konfirmasi
- Halaman Hasil Pendaftaran
- Detail Activity (tambahan)

## 🎨 UI/UX
- Menggunakan Material Design
- Warna konsisten
- Layout rapi dan tidak default
- Custom drawable dan styling

## ⚙️ Teknologi
- Kotlin
- Android Studio
- Material Components
